# BUGFIX 24 – WinMM Joystick API 替換為 DirectInput8 + XInput

**日期**：2026-03-02  
**版本**：AprNes  
**分類**：Refactor / Bug Fix  

---

## 動機

原本使用 WinMM (`joyGetPos` / `joyGetDevCaps`) 作為搖桿輸入 API，此 API 過於老舊：
- 只支援最多 6 軸 + 少量按鈕
- 無法正確識別 XInput 手把（Xbox 系列）
- 無 D-pad / POV hat 原生支援

重構目標：使用 **DirectInput8**（相容 non-XInput 手把）+ **XInput**（Xbox/XInput 手把）雙模式。

---

## 問題一：AccessViolationException（DirectInput COM 互操作）

### 現象

```
System.AccessViolationException: 嘗試讀取或寫入受保護的記憶體。
  於 NativeTools.IDirectInput8W.EnumDevices(...)
  於 NativeTools.DirectInputNative.EnumJoysticks(...)
```

### 根本原因

使用 `[ComImport]` + `[InterfaceType(ComInterfaceType.InterfaceIsIUnknown)]` 建立 COM RCW 時，CLR 在 Marshal managed delegate（`DIEnumDevicesCallback`）作為 COM method 參數時處理有誤，在呼叫 `EnumDevices` 進入原生 vtable 前即發生記憶體存取違規。

### 修復

完全移除 `[ComImport]` 介面，改用 **raw vtable dispatch**：

```csharp
// 讀取 vtable
IntPtr vtable = Marshal.ReadIntPtr(comPtr);
IntPtr fnPtr  = Marshal.ReadIntPtr(vtable, slot * IntPtr.Size);
var fn = Marshal.GetDelegateForFunctionPointer(fnPtr, typeof(T)) as T;

// callback 以 IntPtr 傳入，不走 CLR delegate marshal
IntPtr cbPtr = Marshal.GetFunctionPointerForDelegate(cb);
fn(pDI, DI8DEVCLASS_GAMECTRL, cbPtr, IntPtr.Zero, DIEDFL_ATTACHEDONLY);
GC.KeepAlive(cb);
```

## 重構內容

### 新增：`AprNes/tool/DirectInputHelper.cs`

- `DIJOYSTATE`（80 bytes）、`DIPROPRANGE`、`DIDATAFORMAT`、`DIOBJECTDATAFORMAT` 結構定義
- vtable slot 常數（`DI8Slot`、`DI8DevSlot`）
- 9 種 `[UnmanagedFunctionPointer(CallingConvention.StdCall)]` delegate 類型
- `DirectInputNative` 靜態類：`CreateDirectInput()` / `EnumJoysticks()` / `OpenDevice()` / `PollDevice()` / `ReleaseDevice()`
- `DfDIJoystick` 靜態格式物件（對應 `DIJOYSTATE`）
- SetupDi 掃描 `IG_` 路徑，排除 XInput 裝置進入 DI 列舉

### 修改：`AprNes/tool/joystick.cs`

- 移除 WinMM polling；以 DirectInput + XInput 雙路徑取代
- `Init(IntPtr hwnd)` 初始化 DI 和 XInput
- `joy_event_captur()` 輸出標準 `joystickEvent` 列表
- POV hat → X/Y 轉換（`PovToXY()`），值固定為 0/32767/65535
- AXIS_NOISE=256 過濾中心雜訊

### 修改：`AprNes/tool/NativeAPIShare.cs`

- 移除 WinMM `joyGetPos` / `joyGetDevCaps` P/Invoke 與相關結構體
