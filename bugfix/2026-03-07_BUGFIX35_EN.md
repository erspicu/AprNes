# BUGFIX35: Arbitrary Sprite Zero + Misaligned OAM + Parallel Test Runner + Debug Log Fix

## Date: 2026-03-07

## Baseline: 174 PASS (legacy tests), 108/136 AccuracyCoin PASS (after BUGFIX34)

## Fix Details

### 1. Arbitrary Sprite Zero (PPU.cs) — AccuracyCoin $0458 PASS
- **Problem**: PrecomputeSprite0Line() hardcoded reading sprite 0 data from OAM[0]
- **Correct behavior**: NES hardware starts sprite evaluation at dot 65, using the current OAMADDR as the starting address. The first sprite processed is treated as "sprite 0", even if it is not at OAM index 0
- **Fix**:
  - Added `sprite0_eval_addr` field
  - At dot 65 (visible scanlines + pre-render scanline 261), saves the current `spr_ram_add`
  - PrecomputeSprite0Line() now uses `sprite0_eval_addr` to calculate addrH/addrL, reading Y/tile/attr/X in sequence
  - Correctly simulates the 4-byte wrapping of sprite evaluation (addrL 0→1→2→3→0 increments addrH)

### 2. Misaligned OAM Behavior (PPU.cs) — AccuracyCoin $045A PASS
- **Problem**: Same as above, and mishandled cases where OAMADDR is not 4-byte aligned
- **Correct behavior**: When OAMADDR is not a multiple of 4 (e.g. $81), sprite evaluation still starts reading the Y coordinate from that address, with addrL beginning at a non-zero value
- **Fix**: Same fix above automatically covers this scenario
- **Note**: sprite0_eval_addr retains the full address (no & 0xFC alignment); addrH/addrL are calculated separately

### 3. Debug Log PID-based Path (CPU.cs)
- **Problem**: All headless emulator instances wrote to the same `emu_debug.log`, causing IOException (file locked) during parallel execution
- **Fix**: dbgInit() now uses `%TEMP%/aprnes_debug_{PID}.log`, unless TestRunner explicitly sets DebugLogPath

### 4. Python Parallel Test Runner (run_tests.py)
- **New**: `run_tests.py` — uses ThreadPoolExecutor to run 174 test ROMs in parallel
- **Effect**: 2m11s → ~19s (10 threads, ~6.7x speedup)
- **Usage**: `python run_tests.py` (default 10 threads), `python run_tests.py -j 4` (4 threads)

## Test Results

- Legacy tests: 174/174 PASS (no regression)
- AccuracyCoin: 110/136 PASS (+2), 25 FAIL, 1 SKIP
- New PASS: Arbitrary Sprite Zero ($0458), Misaligned OAM Behavior ($045A)
