AprNes Benchmark ROM
====================

The ROM file(s) in this directory are used exclusively as benchmark test ROMs
for measuring the performance of the AprNes emulator. They are not distributed
for gameplay or commercial purposes.

These ROMs were collected from the nes-test-roms archive:
  https://github.com/christopherpow/nes-test-roms

All copyrights and intellectual property rights remain with the original
authors of each ROM. No ownership is claimed by the AprNes project.
